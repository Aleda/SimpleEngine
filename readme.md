# C++/OpenGL 梦幻西游

个人学习练手项目

### Mac编译指南

1. 安装好git和cmake,然后敲命令
		
		git clone --recursive https://github.com/oceancx/SimpleEngine
		cd SimpleEngine
		mkdir build
		cd build
		cmake ..
		make -j8

2. 下载资源压缩包：[百度云盘](https://pan.baidu.com/s/1o7Gn0t8)，直接在SimpleEngine/下解压即可

3. 然后：

		cd build/
		./SimpleEngine
